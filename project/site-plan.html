WDD 131 Final Project Site Plan - George Olwal

Site Name

NutriCrop Calculator

This name was chosen because it clearly communicates the main function of the website - calculating nutrients for crops. It's concise, memorable, and relevant to the project's purpose.

Site Purpose

NutriCrop Calculator serves as a comprehensive tool for farmers to:

  • Calculate required nutrients and water based on crop recipes using the mole system.
  • Schedule fertilizer applications.
  • Access information on best farming practices.
  • Learn about crop characteristics and fertilizer properties.
  • Assist in farm input budgeting.

The site aims to promote precision farming methods, improve crop yields, and contribute to food security.

Scenarios

  1. How can I determine the exact amount of nutrients my tomato crop needs based on my soil type and local climate?
  2. What's the optimal fertilizer application schedule for my maize crop throughout its growth cycle?
  3. How do I calculate the right balance of locally available fertilizers to meet my crop's nutrient requirements?

The primary color (hsl(212, 90%, 50%)) will be used for main headings and interactive elements like buttons.

The secondary color (hsl(0, 0%, 98%)) serves as the background color for the site, providing a clean and neutral base.

The text color (hsl(0, 0%, 20%)) ensures good readability against the light background.

Typography

Headings: Roboto, sans-serif

Body Text: Open Sans, sans-serif

This combination provides a modern look while ensuring readability across devices.
